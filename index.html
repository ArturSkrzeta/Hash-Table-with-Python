Hash Table Implementation in Python

Intro

Data structure that maps a key to a value in the memeory slot. It uses hash function that converts a key into a hashed code wich is used for the index computing. Received index indicates a memeory slot where we can get a value from.

Features

App includes following features:

  • OOP

Demo

Key-Value inserting with Hash Function:
  1. Each key from a key-value pairs is being converted into a hash code.
  2. Calculating modulo: hash(key) MOD hash_table_size
  4. The outcome of modulo gives a hash_table index where value can be inserted.
  5. In case when the received hash_table index is already occupied by another value, then we have a collision.
  6. In other words, collsion appears when hash function and further modulo generates the same index for more than one key.
  7. When we have a collision, then we need to proceed with additional computation with e.g. double hashing
